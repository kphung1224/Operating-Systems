One function of the operating system is that you have power button that you can press to turn it on. All kinds of devices such like computers or mobile devices have a power button. Without the power button we would have no ability to turn on the device to start the operating system.
Another function of the operating system is that they include various power options. There is a sleep mode where you turn off unecesscary programs and run at a low power mode to conserve battery life. Another mode is hibernate saves all current running programs to the interal hard drive before powering off the device.
The operating system also has a memory management function. It basically allows you use the optimize use of your device's internal memory. With this distributes which program has the highest priority for memory usage and allows the program to run smoother than the rest.
